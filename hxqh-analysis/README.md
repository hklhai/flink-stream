# windows配置hadoop.home.dir
1. 下载hadoop_dll2.6.0包括hadoop.dll，hdfs.dll，winutils.exe，zlib1.dll
2. 解压hadoop.tar.gz
3. 将1中的所有文件拷贝至hadoop/bin下
4. 将hadoop.dll，winutils.exe拷贝至C:\Windows\System32下
